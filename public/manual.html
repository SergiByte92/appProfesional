Manual de ayuda

App Profesional - guia de uso para version web y Android (Capacitor).

1. Descripcion de pantallas

Inicio

Vista de bienvenida con accesos rapidos a las secciones principales y resumen visual del sistema.

Listado

Muestra todos los registros con su estado, categoria y responsable. Desde aqui se abre cada detalle.

Detalle

Presenta la informacion completa de un registro: datos clave, fecha, importe y observaciones.

Informe

Panel de analisis con filtros, KPIs, grafico por categoria y tabla resumen de resultados.

2. Flujo de uso

  1. Entrar en Listado para revisar incidencias.
  2. Abrir Detalle para validar un registro concreto.
  3. Ir a Informe para filtrar y analizar metricas.
  4. Consultar Ayuda para resolver dudas funcionales.

3. Instalacion Android (APK)

Comandos recomendados:

npm run build

npx cap sync android

npx cap open android

Instalacion en emulador/dispositivo:

adb install -r android/app/release/app-release.apk

4. Configuracion basica

next.config.ts: output = "export"

capacitor.config.ts: webDir = "out"

src/app/detalle/[id]/page.tsx: uso de generateStaticParams() para export estatico.
