Manual de usuario

Version web y Android (Capacitor)

1. Descripcion de pantallas

Inicio

Punto de entrada con resumen del flujo de trabajo y acceso rapido a listado, informe y ayuda.

Listado

Muestra todos los registros y su estado. Desde aqui se abre cada detalle.

Detalle

Visualiza la informacion completa de un registro: categoria, responsable, fecha, importe y notas.

Informe

Incluye filtros, KPIs, grafico por categoria y tabla resumen de resultados filtrados.

Centro de ayuda

Contiene este manual y la ayuda contextual por pantalla.

2. Flujo de uso

  1. Entrar en Listado para revisar registros.
  2. Abrir Detalle para inspeccionar un caso concreto.
  3. Usar Informe para analizar datos con filtros y metricas.
  4. Consultar Ayuda para documentacion y dudas de operacion.

3. Instalacion Android

Comandos basicos:

npm run build

npx cap sync android

npx cap open android

Para instalar APK en emulador:

adb install -r android/app/release/app-release.apk

4. Configuracion basica

Archivo de salida para Android:

capacitor.config.ts -> webDir: "out"

Compilacion web estatica:

next.config.ts -> output: "export"

Ruta dinamica exportada:

src/app/detalle/[id]/page.tsx -> generateStaticParams()
